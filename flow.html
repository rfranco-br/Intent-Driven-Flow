SDD path

Teams not ready for autonomous decomposition can substitute a human-authored Cycle Spec for the Orchestrator's planning step. The PO writes intent as normal. A Craft Engineer authors the Cycle Spec — a pre-sequenced task list with dependencies encoded in writing. The Orchestrator executes the spec rather than generating a plan. Everything downstream is identical. Both paths are fully IDF.
