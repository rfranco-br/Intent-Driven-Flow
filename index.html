Step 01 — Intent
Set a goal, not a task list.
The Product Owner writes one outcome statement: what needs to be true for the client, and how you'll know it worked. No Jira tickets. No implementation instructions.
Example: Reduce checkout abandonment at the payment step below 12%. Currently at 22%.
🤝
Step 02 — Gate 1
Align before anyone builds anything.
A short sync: is this the right goal? Is the scope coherent? Every person on the team agrees before execution starts. If it's not clear, it gets rewritten — not estimated.
Nothing starts until Gate 1 passes. No exceptions.
🤖
Step 03 — Execution
Agents write the code. Humans watch the direction.
AI agents execute — planning, coding, testing — within whatever model fits the team: autonomous, hybrid, or spec-driven. The human role is governance: watching for drift, not typing. The feature ships behind a flag set to OFF.
The flag being OFF means it's deployed to production but invisible to users — until a human decides to release it.
🔍
Step 04 — Gate 2
Read the report. Then actually use it.
An automated report is generated: test results, security, performance. A human reads it — then opens the feature and uses it. Both must pass: no technical issues, and it genuinely serves the goal. Anything off gets written as feedback. Agents fix it. The loop runs until both conditions are met.
Automated tests can't tell you if three taps feel like too many. That's what the hands-on review is for.
🚦
Step 05 — Gate 3
One human, one decision: flip the switch.
The Product Owner reads the final report and authorizes the release. The feature flag flips from OFF to ON. Users can now see the feature. This closes the delivery cycle — but not the goal.
Deployment (behind the flag) and release (flipping the flag) are always two separate decisions.
📈
Step 06 — Monitoring
Watch the real number. Close when it moves.
The goal isn't closed when the feature ships — it's closed when the outcome is confirmed. The PO watches the signal. If it hits the target, the intent is marked complete. If not, a new cycle starts under the same goal.
Example: Abandonment drops from 22% → 9.4%. PO confirms. Intent closed.
